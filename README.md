\# AI Transcriber



An offline desktop audio transcription tool built with Python, Faster-Whisper and CustomTkinter.



\## Features



\* GPU accelerated transcription using CUDA

\* Faster-Whisper Large-v3 model

\* Automatic language detection

\* Timestamped transcript generation

\* Metadata export (JSON)

\* Simple desktop GUI

\* Standalone Windows executable



\## Tech Stack



\* Python

\* Faster-Whisper

\* CustomTkinter

\* CUDA

\* PyInstaller



\## Usage



1\. Launch the application.

2\. Select an audio file.

3\. Click \*\*Start Transcription\*\*.

4\. Wait for processing to complete.

5\. Find results in the output folder.



\## Output



\* Transcript (`.txt`)

\* Metadata (`.json`)
